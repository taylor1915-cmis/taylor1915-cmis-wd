Procedure to Create and View a Website on the Production Server

  1. Open Terminal
  2. Make sure your in your repository folder (cd username), and enter in the command "python -m SimpleHTTPServer" This will start the development server.
  3. Create your html folder by typing "gedit 'filename.html'
  4. Now to google and type in "localhost:8000/filename.html" You won't be able to see any changes unless you have done some editing and saving to the html folder.
  5. Start editing your folder, and remember the html format. Copy and paste the template into your file, and add in or change your information. Remember that everytime you make a change to your html folder, you must save it or else nothing will change.
  6. Everytime you make a change and you save, refresh your page, and your changes should appear.
  7. To style your html page, you need to open up a css file. To do this you must put in the command: "gedit 'filenameofhtml.css'
  8. Go back to your html folder and under 'meta charset' put in the link 'link rel="stylesheet" href="filename.css" type="text/css"
  9. Now you can start editing your html on the css folder. Remember to style a certain part of your html you must put in the name of the section and the bracket "{" and to close it put in the bracket "}" after every styling, remember that you must put a semicolon at the end, otherwise the styles after will not appear.
  10. Once you like what you have done you must save both your html and css to your repository folder with the commands "git add, git commit, and git push"
  11. To view it on the production server you put in the command: "sftp username@cmis-hsteacher" It will then ask you for your password.
  12. It will ask you if you want to connect, click the yes, and it will then say you have been connected to the production server.
  13. Make sure your in the public folder so put in the command: "cd" and click public.html. This will make put you into the public folder.
  14. Now type in the command: put 'filename" and your done.
